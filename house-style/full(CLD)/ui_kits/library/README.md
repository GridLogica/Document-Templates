# Library UI kit — GridLogica

Interactive recreation of the GridLogica research library website, derived from the canonical `house-style/styleguide.html`.

## Files

- `index.html` — interactive entry point.
- `components.jsx` — modular components (Wordmark, TopNav, PublicationCard, FocusCard, SearchInput, Hero, Footer, Tag, Buttons).
- `app.jsx` — composed screens: Library, Standards, About, Contact + publication modal.

## Screens

1. **Library** — hero with grid motif, four focus area cards, searchable publication grid.
2. **Standards** — long-form list of all numbered standards.
3. **About** — bio + meta facts.
4. **Contact** — single CTA.
5. **Publication modal** — opens on card click.

## Interactions

- Top nav switches screens.
- Search filters publications by title, code, description, or tag.
- Click any publication card to open a modal preview with the PDF download CTA.
